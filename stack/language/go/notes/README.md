# Go Notes

## 目的

`notes/` は、Go に関する記事素材・学習メモ・一覧向け抜粋を整理する場所。

## 構成

- `listing-summary.md`
  - 言語一覧ページやトグル用の短い要約
- `workflow.md`
  - 実装や検証のときの作業メモ
- `beginner/`
  - 初学者向けの記事素材
- `intermediate/`
  - 中級者・開発者向けの記事素材

## 使い分け

### 一覧ページ向け

- `listing-summary.md`

### 初学者向け Go ページ

- `beginner/language-page.md`

### 中級者向け Go ページ

- `intermediate/syntax-intro.md`
- `intermediate/python-rust-comparison.md`
- `intermediate/use-cases.md`
