# 作業中一覧

## 目的

誰がどのブランチで何を進めているかを、Issue を開かなくてもすぐ確認できるようにするための一覧。

## 使い方

- 作業開始時に1行追加する
- ブランチ名は実際の作業ブランチと一致させる
- マージまたは中断時に `status` と `notes` を更新する
- 役割が重なりそうな変更に入る前にこの一覧を確認する
