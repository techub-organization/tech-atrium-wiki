# Directory Map

## Top Level

- `docs`: 要件、設計、運用ドキュメント
- `backend`: 共同開発者ごとのバックエンド実装雛形
- `language`: 言語別のサンプル、テンプレート、補助ツール
- `framework`: フロントエンドや UI 関連の実装雛形
- `database`: ローカル検証用スキーマ、マイグレーション、シード

## Conventions

- 入口説明は `README.md`
- 草案やテンプレートは `templates`
- 作業メモは `notes`
- 実行可能な雛形は `project`
- 検証系は `tests` または `migrations` / `seeds`
